# Space wars
A tiny terminal game written with asynco Python.

## Requirements
- Python 3.8 +
- curses (built-in on Unix systems). On Windows:
```
pip install windows-curses
```

## Running
```
python3 game.py
```

### controls
| Key	 | Action |
|----------|----------|
| ← ↑ ↓ →  | Move spaceship |
| Ctrl-C   | Quit  |
